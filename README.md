# Zephyrous

## 游戏介绍

### 游戏类别

- 2D 像素沙盒生存游戏

### 核心玩法

- 随机生成的世界（地形、洞穴、生物群落）
- 资源采集与合成系统
- 动态昼夜循环与天气系统
- 角色属性与装备系统
- NPC 交互与事件触发

## 技术栈
- SFML - 图形渲染，音频管理
- SQlite json - 数据存储和配置文件
- Box2d - 物理碰撞检测
- SFML - 音频管理
- ENet 或 RakNet 多人联机支持(2.0)
- 项目管理(CMakeLists)
